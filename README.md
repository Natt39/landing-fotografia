# LuzYEnfoque — Landing Page

## Descripción del proyecto
Landing Page para un emprendimiento de fotografía profesional,
desarrollada como evaluación práctica de la asignatura de
Desarrollo Front End en INACAP.

## Problemática abordada
Un emprendimiento local de fotografía necesitaba una página web
simple para promocionar sus servicios y captar nuevos clientes.

## Tecnologías utilizadas
- HTML5 (etiquetas semánticas)
- CSS3 (flexbox, media queries, animaciones)
- Tailwind CSS (framework de estilos)
- Git y GitHub (control de versiones)

## Uso de Inteligencia Artificial
Se utilizó GitHub Copilot como herramienta de apoyo en dos momentos:
1. Generación de la estructura del formulario de contacto
2. Sugerencia de la estructura de tarjetas de servicios

En ambos casos el código fue revisado, comprendido y adaptado
al proyecto. Los usos están marcados con comentarios en el HTML.
